Shell grows to 92 px. Each workspace tab gains a 16 px line icon. Active tab has a green-tinted fill, brighter icon, and a glowing 3 px green underline. The current page reads at a glance.
P Two personas, one UI
Standard operator
Recording session — one click, then leaves
  1. Opens app → arrives in Lighting workspace.
  2. Glances at the wall of scene tiles on the left.
  3. Clicks the right tile (e.g. Demo bench). Fixtures emit, active pill confirms.
  4. Moves to the recording workspace. Done.
Senior operator
Setup & commissioning — many edits, same canvas
  1. Patches new fixtures via P overlay: assigns DMX, rig height, beam angle, fires Identify. (DMX peek surfaces here.)
  2. Drags fixtures on the plot to their spatial positions; clicks one to edit intensity / CCT in the inspector.
  3. Saves the current state as a new scene (+ New scene) or updates an existing one (drift → Save).
  4. Manages groups via the bottom strip + inspector Group tab.
